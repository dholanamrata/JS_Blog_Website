What is a Console/Terminal/Shell?

Before discussing a Console/Terminal/Shell let me tell you they all are different but have similar functionalities and operations. So we’ll discuss them separately and see how they are linked together.

Console

A CONSOLE means a combination of a display monitor and input devices, usually a keyboard and mouse pair, which allows a user to input commands and receive visual output from a computer. The console is nothing but a combination of devices that allows input/output operation. Another term for console is terminal. The term console usually refers to a terminal attached to a minicomputer or mainframe and used to monitor the status of the system.

Terminal

The word TERMINAL is a unix terminology, it means a particular kind of special file(or device file) that is capable of implementing additional commands (beyond read & write). A terminal is an interface to the underlying OS i.e, a combination of screen+keyboard which is connected to a server. It is also called TeleTYpewriter (tty).
